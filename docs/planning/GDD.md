# Game Design Document (GDD)
## 에테리아의 균열 (Crack of Eteria)

**버전:** 0.2
**날짜:** 2026-04-20
**상태:** 확정 (RPG+탐험 혼합 방향 반영)

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 제목 | 에테리아의 균열 (Crack of Eteria) |
| 장르 | 탑다운 2D 실시간 액션 RPG (RPG+탐험 혼합) |
| 플레이어 수 | 싱글플레이어 |
| 플레이 시간 | 5~10시간 |
| 플랫폼 | 웹 브라우저, Android (WebView) |
| 엔진 | Phaser.js 3.x + TypeScript |

### 1.1 핵심 경험

> "평범한 청년이 세계를 구하는 영웅이 되는 성장 이야기"

플레이어는 봉인이 풀리기 시작한 마왕의 위협 속에서 4개 지역을 탐험하며 성장한다. 근접/원거리 전투 선택, 간결한 퀘스트 흐름, 뚜렷한 보스 패턴으로 가볍지만 임팩트 있는 인디 RPG를 지향한다.

RPG의 레벨업·스탯 성장과 젤다 방식의 **특수 아이템 게이트**를 결합한다. 보스를 처치하면 특수 아이템을 획득하고, 그 아이템이 새로운 경로나 퍼즐을 열어주는 방식으로 탐험에 동기를 부여한다.

## 2. 스토리

### 2.1 배경 세계

에테리아는 마법 결정체(에테르)가 자연에 퍼진 판타지 세계다. 수백 년 전 용사들이 마왕 발코르를 고대 성채에 봉인했으나, 에테르 고갈로 봉인이 서서히 약해지고 있다. 어둠의 마물들이 다시 땅으로 흘러나오기 시작한다.

### 2.2 주인공

**아이든 (Aiden)**
- 하벤 마을 출신의 20대 청년
- 부모는 에테르 탐험 중 실종 (후반부 복선)
- 무기 선택 가능: 검(근접) 또는 단검+마법서(원거리)
- 성격: 과묵하지만 정의감 강함

### 2.3 핵심 NPC

| NPC ID | 이름 | 역할 | 위치 |
|--------|------|------|------|
| npc_elder | 장로 에릭 | 메인 퀘스트 진행자, 마을 안내 | 하벤 마을 |
| npc_blacksmith | 대장장이 핸 | 무기 강화, 사이드 퀘스트 | 하벤 마을 |
| npc_merchant | 상인 리나 | 아이템 구매/판매 | 하벤 마을 |

| npc_scholar | 학자 토마스 | 고대 유적 관련 정보 | 하벤 마을 |
| npc_guard_captain | 경비대장 레온 | 사이드 퀘스트, 전투 팁 | 하벤 마을 |

| npc_survivor | 생존자 카일 | 용암 동굴 직전 사이드 퀘스트 | 고대 유적 출구 |

### 2.4 스토리 흐름 요약

```
하벤 마을 (평화 일상)
  → 마물 출현 사건 발생
  → 장로에게 경위 청취 → 어둠의 숲 탐험 지시
어둠의 숲
  → 고블린 세력 격파 → 고블린 왕 처치
  → 고대 열쇠 단서 발견
고대 유적
  → 리치의 봉인 해제 흔적 조사
  → 리치 처치 → 마왕 부활 임박 확인
용암 동굴
  → 마왕의 전위대 격파 → 화염 드래곤 처치
  → 마지막 에테르 결정체 수집
마왕의 성
  → 어둠의 기사 격파 → 성 심부 돌파
  → 마왕 발코르 최후 결전
엔딩
  → 봉인 복원 성공 → 에테리아 평화 회복
```

## 3. 월드 및 지역

### 3.1 지역 목록

| 지역 ID | 이름 | 타입 | 분위기 |
|---------|------|------|--------|
| scene_haven | 하벤 마을 | 마을 (안전) | 평화로운 농촌 |
| scene_forest | 어둠의 숲 | 던전 | 어둡고 울창한 숲 |
| scene_ruins | 고대 유적 | 던전 | 황폐한 고대 문명 |
| scene_cavern | 용암 동굴 | 던전 | 붉은 용암과 암석 |
| scene_fortress | 마왕의 성 | 최종 던전 | 어두운 성채 |

### 3.2 지역 상세

#### 하벤 마을 (Haven Village)
- **역할**: 시작 지점, 안전지대, 상점/NPC 허브
- **맵 크기**: 40×40 타일
- **주요 포인트**: 장로의 집, 대장간, 상점, 치유소, 마을 광장
- **이동 제한**: 마물 미출현, 전투 없음

#### 어둠의 숲 (Dark Forest)
- **역할**: 첫 번째 전투 지역
- **맵 크기**: 80×80 타일 (2개 구역)
- **주요 포인트**: 숲 입구, 고블린 캠프, 보스 구역(나무 왕좌)
- **주요 적**: 고블린, 고블린 궁수, 늑대, 거미
- **보스**: 고블린 왕 → 처치 시 **숲의 나침반** 획득
- **던전 퍼즐**: 압력 스위치 (정해진 순서로 밟으면 문 개방)
- **잠긴 구역**: 나침반 없이는 진행 불가한 숨겨진 경로 존재

#### 고대 유적 (Ancient Ruins)
- **역할**: 미스터리/스토리 중심 지역
- **맵 크기**: 80×80 타일 (3개 구역)
- **주요 포인트**: 유적 입구, 함정 지대, 영혼 제단, 보스 방
- **주요 적**: 스켈레톤, 스켈레톤 궁수, 좀비, 흑마법사
- **보스**: 리치 → 처치 시 **고대 갈고리** 획득
- **던전 퍼즐**: 블록 밀기 (돌 블록을 홈에 밀어 넣어 함정 비활성화)
- **잠긴 구역**: 갈고리 없이는 건널 수 없는 간격이 넓은 발판 구역

#### 용암 동굴 (Lava Cavern)
- **역할**: 고난이도 지역, 전투 집중
- **맵 크기**: 60×100 타일 (수직형, 3개 구역)
- **주요 포인트**: 동굴 입구, 용암 지대(낙하 데미지), 드래곤 둥지
- **주요 적**: 박쥐, 화염 슬라임, 가고일, 용암 골렘
- **보스**: 화염 드래곤 → 처치 시 **화염 방패** 획득
- **던전 퍼즐**: 횃불 점화 (올바른 순서로 점화 시 잠긴 문 개방)
- **잠긴 구역**: 화염 방패 없이는 통과 불가한 용암 통로

#### 마왕의 성 (Dark Fortress)
- **역할**: 최종 지역, 클라이맥스
- **맵 크기**: 60×80 타일 (4개 구역)
- **주요 포인트**: 성문, 병영, 왕좌 전실, 왕좌의 방
- **주요 적**: 오크, 트롤, 슬라임, 흑마법사, 다크 나이트(일반 적 버전)
- **중간 보스**: 어둠의 기사
- **최종 보스**: 마왕 발코르

## 3.3 특수 아이템 게이트 시스템

보스를 처치하면 **특수 아이템(키 아이템)**을 획득한다. 이 아이템은 이전 지역의 잠긴 구역을 열어주어, 클리어 후 되돌아와 탐험할 동기를 제공한다.

| 특수 아이템 ID | 이름 | 획득 | 효과 |
|--------------|------|------|------|
| `key_compass` | 숲의 나침반 | 고블린 왕 처치 후 드롭 | 숲 내 숨겨진 경로 표시, 나침반 게이트 통과 가능 |
| `key_hookshot` | 고대 갈고리 | 리치 처치 후 드롭 | 갈고리 걸리는 오브젝트에 발사 → 당기거나 이동 |
| `key_flameshield` | 화염 방패 | 화염 드래곤 처치 후 드롭 | 용암 통로 5초간 피해 없이 통과 가능 (쿨다운 30초) |

### 게이트 규칙

- 게이트는 **물리적 장벽 타일**로 시각화 (나침반: 덤불 벽, 갈고리: 넓은 틈, 화염 방패: 용암 통로)
- 특수 아이템 미보유 시 플레이어가 접근하면 힌트 텍스트 표시: *"무언가가 있으면 통과할 수 있을 것 같다."*
- 키 아이템은 분실되지 않음 — 한 번 획득하면 영구 보유

### 탐험 선택지 — 하트 조각

선택 탐험 구역(잠긴 구역, 숨겨진 길)에는 **하트 조각**이 배치된다.

| 규칙 | 내용 |
|------|------|
| 필요 수량 | 4개 = HP +100 영구 증가 |
| 총 배치 수 | 8개 (최대 2회 HP 증가 가능) |
| 획득처 | 각 지역 잠긴 구역 보상, 보스 방 이전 선택 경로 |
| 강제성 | 전혀 없음 — 수집하지 않아도 메인 스토리 완료 가능 |

## 3.4 던전 퍼즐 시스템

각 던전마다 고유한 퍼즐 메카닉을 1종 배치한다. 퍼즐 해결 시 잠긴 문 개방 또는 하트 조각 획득.

| 던전 | 퍼즐 타입 | 메카닉 설명 |
|------|----------|------------|
| 어둠의 숲 | 압력 스위치 | 맵에 배치된 스위치를 지정 순서대로 밟으면 문 개방. 순서 틀리면 리셋. |
| 고대 유적 | 블록 밀기 | 무거운 돌 블록을 홈(홈 타일)에 밀어 넣으면 함정 비활성화 또는 문 개방. |
| 용암 동굴 | 횃불 점화 | 방 내 횃불을 올바른 순서로 점화. 틀리면 전체 꺼지며 리셋. |
| 마왕의 성 | 거울 반사 | 마법 거울을 회전시켜 빛줄기를 수정에 조준하면 봉인 해제. |

### 공통 규칙
- 퍼즐은 전투 없이 단독으로 도전 가능
- 해결 실패 시 패널티 없음 — 무제한 재시도
- 퍼즐 완료 상태는 저장 데이터에 기록 (재방문 시 재풀이 불필요)

## 4. 플레이어 시스템

### 4.1 기본 스탯

| 스탯 | 시작 값 | 최대 값 (레벨 20) | 설명 |
|------|---------|-------------------|------|
| HP | 100 | 400 | 0이 되면 게임오버 |
| MP | 50 | 200 | 마법 사용 시 소모 |
| ATK | 15 | 80 | 물리 공격력 |
| DEF | 10 | 60 | 피해 감소량 |
| SPD | 150 | 220 | 이동 속도 (px/s) |

### 4.2 레벨 시스템

- **레벨 범위**: 1~20
- **경험치 획득**: 적 처치, 퀘스트 완료
- **레벨업 보너스**: HP +15, MP +5, ATK +3, DEF +2
- **경험치 테이블**: 레벨 n 달성에 필요한 누적 경험치 = `50 * n^1.5` (반올림)

| 레벨 | 누적 EXP |
|------|----------|
| 2 | 71 |
| 5 | 559 |
| 10 | 1581 |
| 15 | 2905 |
| 20 | 4472 |

### 4.3 무기 선택 (게임 시작 시 1회)

| 선택 | 기본 공격 | 특수 공격 |
|------|----------|----------|
| 검사 (Swordsman) | 근접 검 공격 (ATK 100%) | 회전베기 (범위, MP 10 소모) |
| 마도사 (Mage) | 원거리 마법탄 (ATK 70%) | 마법 폭발 (범위, MP 20 소모) |

### 4.4 대시 (회피)

- **발동**: 방향키 + 대시 버튼
- **무적 프레임**: 0.25초
- **쿨다운**: 1.5초
- **이동 거리**: 120px

## 5. 전투 시스템

### 5.1 데미지 계산

```
피해량 = 공격자 ATK - 방어자 DEF / 2  (최소 1)
```

### 5.2 공격 타입

| 타입 | 범위 | 발동 방식 |
|------|------|----------|
| 근접 공격 | 전방 60px 반원 | 공격 버튼 |
| 원거리 공격 | 직선 발사체 (최대 300px) | 공격 버튼 |
| 특수 공격 | 주위 80px 원형 | 특수 버튼 + MP 소모 |

### 5.3 적 AI 상태

| 상태 | 조건 | 행동 |
|------|------|------|
| 순찰 | 플레이어 미감지 | 웨이포인트 이동 |
| 감지 | 감지 반경 내 진입 | 알림 이펙트 후 추적 전환 |
| 추적 | 플레이어 감지됨 | 플레이어 방향으로 직선 이동 |
| 공격 | 공격 사거리 진입 | 공격 애니메이션 + 데미지 판정 |
| 귀환 | 추적 이탈 반경 초과 | 순찰 위치로 복귀 |

#### 보스 패턴 상세

**고블린 왕**
1. 기본 근접 연타 (3회 콤보)
2. 소환 (고블린 2마리) — HP 50% 미만 시 사용

1. 마법탄 3발 방사

## 9. 밸런싱 가이드

### 9.1 권장 레벨 진행

| 지역 | 권장 레벨 | 예상 플레이 시간 |
|------|----------|----------------|
| 하벤 마을 | 1 | 15~20분 |
| 어둠의 숲 | 2~5 | 1~1.5시간 |
| 고대 유적 | 6~10 | 1.5~2시간 |
| 용암 동굴 | 11~15 | 1.5~2시간 |
| 마왕의 성 | 16~20 | 1.5~2시간 |

### 9.2 골드 밸런싱

- 일반 적 드롭: 5~30G
- 보스 드롭: 100~300G
- 소형 포션: 50G, 중형: 150G, 대형: 500G
- 무기 구매: 200~800G
- 방어구 구매: 150~600G

### 9.3 난이도 조정 원칙

- 플레이어가 지역 권장 레벨 ±2 이내라면 클리어 가능해야 한다
- 보스 전투는 패턴 파악 후 3~5회 이내 클리어 가능한 난이도
- 게임오버 시 마지막 세이브 포인트(지역 입구)에서 재개

## 10. UI/UX 설계

### 10.1 HUD (전투 중)

- 좌상단: HP 바, MP 바
- 우상단: 레벨, 현재 EXP
- 하단 중앙: 스킬/아이템 단축키 슬롯 (4칸)
- 미니맵: 우하단 (옵션)

### 10.2 메뉴 화면

- 인벤토리 (I키 또는 버튼)
- 퀘스트 일지 (Q키 또는 버튼)
- 세이브/불러오기 (마을에서만 가능)

### 10.3 대화 시스템

- 하단 대화창 (화면 하단 20%)
- NPC 이름 + 초상화 + 대사 텍스트
- 클릭/버튼으로 다음 진행
- 분기 없음 (M3 범위)
