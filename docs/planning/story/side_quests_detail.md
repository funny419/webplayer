# 사이드 퀘스트 상세 기획

**연계 파일:** `docs/data/quests.json`, `docs/data/dialogues.json`
**날짜:** 2026-04-20

---

## 개요

사이드 퀘스트 7개는 모두 하벤 마을 NPC에게서 시작한다. 메인 퀘스트 진행에 독립적이지만 특정 메인 퀘스트 완료 후에만 수락 가능한 것들이 있다. 보상은 경험치·골드·아이템으로 구성되며, 일부는 시스템 unlock을 포함한다.

---

## sq_01 — 사라진 아이

| 항목 | 내용 |
|------|------|
| 의뢰자 | 소녀 리아 (`npc_child`) |
| 수락 조건 | `mq_02` 완료 후 |
| 목표 | kill × 2 + location |
| 보상 | EXP 60, 부활초 1개 |
| 플레이 시간 | 약 10분 |

### 스토리
마을 외곽에서 혼자 울고 있는 소녀 리아는 오빠가 약초를 캐러 숲에 들어갔다가 돌아오지 않는다며 도움을 요청한다. 숲 입구 근처 고블린 2마리를 처치하고, 서쪽 오래된 참나무 옆 구출 지점에 도달하면 오빠가 숨어있는 걸 발견한다.

### 연출 포인트
- 구출 지점 도달 시 짧은 연출 텍스트: *"나무 뒤에서 떨고 있는 소년을 발견했다."*
- 퀘스트 완료 후 리아 idle 대사 변경: *"오빠가 돌아왔어요! 아저씨 덕분이에요."*

### 설계 메모
- `kill` 목표 선완료 필요 없음 — `location` 먼저 도달해도 `kill` 목표가 미완이면 퀘스트 완료 불가 (objectives 순서 강제 처리)
- 실제 구현은 kill 완료 후 location 활성화가 자연스럽지만, 어느 쪽이든 둘 다 완료 시 퀘스트 종료

## sq_02 — 대장장이의 부탁

| 항목 | 내용 |
|------|------|
| 의뢰자 | 대장장이 핸 (`npc_blacksmith`) |
| 수락 조건 | `mq_01` 완료 후 |
| 목표 | collect × 3 (철광석) |
| 보상 | EXP 80, 대장간 강화 메뉴 unlock |
| 플레이 시간 | 약 15분 |

### 스토리
대장장이 핸은 무기 제련에 필요한 철광석이 떨어졌다고 고민한다. 숲이 위험해져서 직접 가기 어렵다며 아이든에게 채취를 부탁한다.

### 철광석 획득처
| 종류 | 위치 | 수량 | 비고 |
|------|------|------|------|
| 광맥 오브젝트 (`object_ore_vein`) | 어둠의 숲 서쪽 바위 지대 3곳 | 각 1개 | 1회 채취 후 소멸 |
| 고블린 드롭 | 어둠의 숲 전역 | 30% 확률 | 광맥 부족 시 대안 |

### unlock 내용: 대장간 강화 메뉴
완료 후 대장장이 핸과 대화 시 "무기 강화" 옵션 추가:
- 철 검 → 은 검 강화: 골드 400G + 철광석 2개
- 철 지팡이 → 마나 지팡이 강화: 골드 400G + 철광석 2개

### 설계 메모
- 퀘스트 완료 전이라도 철광석이 인벤토리에 있으면 대장간 강화 불가 (퀘스트 아이템으로 취급)
- 완료 후 철광석은 일반 아이템으로 전환되어 인벤토리에 남지 않음 (전량 납부)

## sq_03 — 치유사의 약초

| 항목 | 내용 |
|------|------|
| 의뢰자 | 치유사 미아 (`npc_healer`) |
| 수락 조건 | `mq_01` 완료 후 |
| 목표 | collect × 5 (치유초) |
| 보상 | EXP 60, 중형 포션 3개 |
| 플레이 시간 | 약 15분 |

### 스토리
치유사 미아는 포션 재료인 치유초가 고갈되어 부상자를 제대로 치료하지 못하고 있다. 숲의 밝은 공터 근처에 치유초가 자라지만 혼자 가기에는 위험하다고 말한다.

### 치유초 획득처
| 종류 | 위치 | 수량 | 비고 |
|------|------|------|------|
| 치유초 오브젝트 (`object_herb_plant`) | 어둠의 숲 중앙 공터 인근 5곳 | 각 1개 | 1회 채취 후 소멸 |
| 거미 드롭 | 어둠의 숲 전역 | 20% 확률 | 보조 수단 |

### 설계 메모
- sq_02와 동일한 시기에 수락 가능 → 동시 진행으로 숲 탐색 효율화 가능
- 치유초 오브젝트와 광맥 오브젝트는 Tiled 오브젝트 레이어에 `type: "collectible"` + `item_id` 속성으로 배치

## sq_04 — 학자의 탁본

| 항목 | 내용 |
|------|------|
| 의뢰자 | 학자 토마스 (`npc_scholar`) |
| 수락 조건 | `mq_05` 완료 후 |
| 목표 | location × 3 (비문 조사) |
| 보상 | EXP 100, 에테르 파편 2개 |
| 플레이 시간 | 약 20분 |

### 스토리
학자 토마스는 고대 마왕 봉인의 공식이 유적 내 비문에 새겨져 있다는 사실을 알고 있다. 세 곳의 비문을 탁본해달라고 요청한다.

### 비문 위치 상세
| ID | 위치 설명 | 지역 |
|----|----------|------|
| `inscription_1` | 유적 입구 외벽 좌측 — 바로 보임 | 유적 구역 1 |
| `inscription_2` | 2층 복도 끝 함정 통과 후 — 조심 필요 | 유적 구역 2 |
| `inscription_3` | 지하 저장고 — 적 다수 배치 | 유적 구역 3 |

### 연출 포인트
- 각 비문 조사 시 텍스트 팝업: *"고대어로 봉인 공식의 일부가 새겨져 있다."*
- 3개 모두 완료 후 학자 대사에서 봉인 공식의 힌트 제공 (스토리 보조)

### 설계 메모
- 비문 오브젝트 Tiled 속성: `type: "interact"`, `dialogue_id: "inscription_1_text"` 등
- 상호작용 키 입력(F 또는 버튼) → QuestSystem에 location 목표 완료 이벤트 발생

## sq_05 — 생존자 구출

| 항목 | 내용 |
|------|------|
| 의뢰자 | 생존자 카일 (`npc_survivor`) |
| 수락 조건 | `mq_07` 완료 후 |
| 목표 | location × 1 (생존자 위치 도달) |
| 보상 | EXP 80, 대형 포션 1개 |
| 플레이 시간 | 약 10분 |

### 스토리
유적 밖에서 만난 카일은 용암 동굴을 탐사하다 동료를 두고 혼자 빠져나왔다. 동료가 고립된 위치를 알려주며 구출을 요청한다.

### 생존자 위치
- 동굴 입구에서 서쪽 샛길 — 메인 경로에서 한 칸 이탈한 좁은 통로
- 생존자 오브젝트 배치: `npc_survivor_friend` (대화 가능 NPC, 일회성)
- 도달 시 짧은 대화: *"당신이 카일이 보낸 사람인가요? 감사합니다!"* → 퀘스트 완료

### 설계 메모
- 생존자는 퀘스트 완료 후 맵에서 제거 (despawn 이벤트 필요)
- 카일은 mq_08 의뢰자이기도 하므로 sq_05 완료 후 카일 idle 대사 교체

## sq_06 — 전설의 무기 조각

| 항목 | 내용 |
|------|------|
| 의뢰자 | 대장장이 핸 (`npc_blacksmith`) |
| 수락 조건 | `mq_08` 완료 후 |
| 목표 | collect × 1 (드래곤 비늘) |
| 보상 | EXP 120, 화염 검 또는 화염 지팡이 (직업별) |
| 플레이 시간 | 화염 드래곤 처치 시 자동 완료 |

### 스토리
대장장이 핸이 오래된 설계도를 꺼내며 말한다: *"전설의 불꽃 무기를 만들려면 드래곤 비늘이 필요해."* 화염 드래곤을 처치하면 드래곤 비늘을 얻을 수 있다.

### 드롭 연계
- `boss_fire_drake` 처치 시 드롭 테이블에 `item_dragon_scale` rate 1.0 설정 (balance.json 참조)
- mq_09와 동일한 보스 처치이므로 메인/사이드 동시 진행 가능

### 보상 분기
```
player.class === "swordsman" → weapon_flame_sword
player.class === "mage"      → weapon_fire_staff
```

### 설계 메모
- sq_06 수락 전에 드래곤 비늘을 획득해도 인벤토리에 보관됨 → 수락 후 즉시 완료 가능
- sq_06 보상 무기와 shop에서 구매 가능한 화염 무기는 동일 아이템 (중복 방지는 인벤토리 수량으로 처리)

## sq_07 — 숨겨진 보물

| 항목 | 내용 |
|------|------|
| 의뢰자 | 경비대장 레온 (`npc_guard_captain`) |
| 수락 조건 | `mq_01` 완료 후 (보물상자는 해당 지역 해금 후 접근 가능) |
| 목표 | location × 4 (지역별 숨겨진 보물상자) |
| 보상 | EXP 200, 귀환석 3개 |
| 플레이 시간 | 게임 전체에 걸쳐 진행 (장기 퀘스트) |

### 스토리
경비대장 레온이 오래된 지도 조각을 꺼내며 말한다: *"선배 탐험가들이 각 지역에 물자를 숨겨뒀다는 기록이 있어. 4곳을 확인해줄 수 있겠나?"*

### 보물상자 위치 상세
| ID | 지역 | 위치 설명 | 접근 조건 |
|----|------|----------|----------|
| `hidden_chest_1` | 어둠의 숲 | 숲 남서쪽 막다른 길 — 일반 경로 이탈 필요 | `mq_02` 이후 |
| `hidden_chest_2` | 고대 유적 | 지하 비밀 통로 끝 — 숨겨진 벽 상호작용 후 접근 | `mq_05` 이후 |
| `hidden_chest_3` | 용암 동굴 | 용암 사이 외딴 섬 — 도약 경로 필요 | `mq_08` 이후 |
| `hidden_chest_4` | 마왕의 성 | 병영 창고 — 일반 적 2마리 처치 후 접근 | `mq_10` 이후 |

### 보물상자 내용물 (퀘스트 보상과 별개)
| 위치 | 내용물 |
|------|--------|
| 어둠의 숲 | 골드 150G + 소형 포션 2개 |
| 고대 유적 | 은 검 / 마나 지팡이 (직업별) |
| 용암 동굴 | 골드 300G + 중형 포션 2개 |
| 마왕의 성 | 어둠의 검 / 공허 지팡이 (직업별) |

### 연출 포인트
- 숨겨진 벽 (`hidden_chest_2` 접근용): 상호작용 키 → *"벽의 일부가 움직인다..."* 텍스트 후 통로 개방
- 용암 섬 접근: 점프 대신 좁은 고정 발판 경로 (낙사 데미지 없음, 접근 실패 시 재시도 가능)

### 설계 메모
- 각 보물상자는 1회 개방 후 저장 데이터에 `events` 배열에 `chest_forest_opened` 등 이벤트 플래그로 기록
- 퀘스트 완료는 4개 location 목표 모두 달성 시 → 레온에게 귀환석 수령
- 장기 퀘스트이므로 퀘스트 일지에 "n/4 발견" 진행도 표시 필요
