# 비주얼 방향성 문서

**프로젝트:** 웹 기반 탑다운 2D 액션 RPG
**날짜:** 2026-04-20
**작성:** 디자인 담당 (DS)

---

## 1. 아트 스타일 개요

### 핵심 방향
- **픽셀 아트 (Pixel Art)** — 32×32px 타일 기반의 레트로 감성
- **다크 판타지 + 온기** — 어두운 던전과 따뜻한 마을이 대비를 이루는 세계관
- 참조 레퍼런스: *Undertale*, *Stardew Valley*, *Enter the Gungeon* (탑다운 시점)

### 타일 규격
| 항목 | 수치 |
|------|------|
| 기본 타일 크기 | 32 × 32 px |
| 스프라이트 단위 | 32 × 32 px (캐릭터), 최대 64 × 64 px (보스) |
| 맵 최대 크기 | 100 × 100 타일 |
| 애니메이션 프레임 | 캐릭터 이동: 4프레임, 공격: 4프레임, 대기: 2프레임 |

## 3. 캐릭터 비주얼 가이드

### 플레이어
- 중성적인 실루엣, 판타지 모험가 복장
- 무기에 따라 시각적으로 구분 (검 = 허리에 착용, 활 = 등에 착용)
- 방어구 업그레이드 시 스프라이트 레이어 교체 방식

### 적 디자인 원칙
- **일반 적**: 단순한 실루엣, 쉽게 식별 가능
- **엘리트 적**: 일반 적보다 1.5배 크기, 다른 색상 강조
- **보스**: 64×64px, 복잡한 애니메이션, 독특한 실루엣

### 색상으로 위협도 표현
| 등급 | 색상 예시 |
|------|---------|
| 약함 (Weak) | 무채색 계열 (회색 슬라임, 갈색 고블린) |
| 보통 (Normal) | 차가운 색 (파란 언데드, 초록 오크) |
| 강함 (Elite) | 따뜻한 경고색 (주황, 빨강) |
| 보스 | 독특한 배색 + 발광 이펙트 |

## 4. UI/UX 디자인 원칙

### HUD (화면 내 상시 표시)
- **최소주의**: 게임 화면을 가리지 않음
- HP/MP 바는 좌측 하단, 소형 바 형태
- 미니맵은 우측 상단 (옵션으로 숨기기 가능)
- 모바일: 하단 가상 버튼 (크기 64px 이상, 터치 인식 오차 허용)

### 메뉴/인벤토리
- 반투명 다크 패널 위에 아이콘 그리드
- 아이콘 크기: 32×32px
- 선택된 아이템: 골드 테두리 강조

### 폰트 방향성
- 제목/헤딩: 픽셀 폰트 (예: Press Start 2P)
- 본문/대화: 가독성 우선 폰트 (예: VT323 또는 Pixel Operator)
- 최소 본문 크기: 16px (모바일 가독성 고려)

## 5. 지역별 타일셋 가이드

### 마을 (Town)
- 따뜻한 색조, 나무 건물, 석조 길
- 조명: 황금빛 가로등, 창문에서 새는 불빛
- 식물, 우물, 간판 등 생활감 있는 소품

### 야외 필드 (World)
- 풀밭, 흙길, 나무, 바위
- 시간대 개념은 없음 (정적 조명)
- 전환 구역에 절벽/물 활용

### 던전 (Dungeon)
- 석재 바닥/벽, 횃불 이펙트
- 함정 타일은 플레이어 접근 전 식별 가능한 시각적 단서 제공
- 보스 방 진입 전: 레드 토치 또는 해골 장식으로 경고

---

## 6. 이펙트 & 애니메이션

### 전투 이펙트
- 기본 공격 히트: 작은 노란/흰 섬광 (2~3프레임)
- 마법 공격: 파란/보라 파티클 (4~6프레임)
- 대시 무적: 잔상 이펙트 (반투명 복사본 3~4개)
- 적 사망: 사망 애니메이션 + 페이드아웃

### 환경 이펙트
- 횃불: 루프 불꽃 애니메이션 (4프레임)
- 물: 웨이브 타일 애니메이션
- 풀: 바람에 흔들리는 애니메이션 (옵션)
