# game-planner (게임 기획) 역량

> 관리: writer

---

**역할:** GDD 작성, 퀘스트/대화/밸런싱 데이터 설계
**담당 레이블:** `planning`

| 분류 | 도구/역량 |
|------|----------|
| 문서 작성 | Markdown, JSON 스키마 설계 |
| 기획 산출물 | GDD, quests.json (18개), dialogues.json (42개), balance.json |
| 담당 경로 | `docs/planning/` 폴더 (스토리, 퀘스트, NPC, 밸런싱) |
| 연동 설계 | Phaser.js QuestSystem/DialogueSystem 연동 데이터 구조 설계 |
| 맵 연동 | Tiled 오브젝트 레이어 속성 정의 (웨이포인트, spawn 포인트) |
| 밸런싱 | 레벨 성장 곡선, 적 스탯/보스 패턴, 경제 시스템(골드/아이템 가격) |

## 주요 산출물

| 파일 | 내용 |
|------|------|
| `docs/planning/GDD.md` | Game Design Document |
| `docs/planning/data/quests.json` | 퀘스트 18개 완전한 형태 |
| `docs/planning/data/dialogues.json` | NPC 10명 대사 42개 |
| `docs/planning/data/balance.json` | 밸런싱 마스터 데이터 |
| `docs/planning/story/side_quests_detail.md` | 사이드 퀘스트 상세 기획 |
